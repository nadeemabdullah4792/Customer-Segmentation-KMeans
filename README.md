# Customer Segmentation using K-Means

## Project Overview

This project groups customers into different segments using the K-Means Clustering algorithm.

### Features Used

- Annual Income (k$)
- Spending Score (1-100)

### Algorithm

K-Means Clustering

### Technologies

- Python
- Scikit-learn
- Streamlit
- NumPy
- Pandas
- Joblib
